About Full Stack Web Development

Full Stack web development involves both front-end and back-end web development.A person who knows front-end languages and front-end framework known as front end web developer and A person who knows the back-end languages and back-end technologies known as backend developer. Full Stack web developers are jack of all trades. Full stack web developer are knowledgeable in every level of how the web work. The developer also have the ability to design complete web application and websites by working on frontend, backend, database, dubugging of web application or website.

Full Stack Development Contains Three Layers :-

1.Presentation Layer :- In this layer developer Design user interface by using the Front-end languages (Ex. HTML, CSS, JAVASCRIPT)
2.Logical Layer :- This layer refers to the server side development of the app, also include the data consistency and manages the database using queries and APIS.
3.DataBaseLayer :- This layer provide the Storage or Space for the data Addition, Deletion, Retrival etc.
